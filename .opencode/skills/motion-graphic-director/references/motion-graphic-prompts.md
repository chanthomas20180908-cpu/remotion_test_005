# Motion Graphic Prompts

---

## Section 1: UI Chaos & Collapse

### 1. Infinite Scrolling Feed Collapse
A modern social feed scrolls smoothly downward — posts, comments, headlines.
Suddenly the scroll accelerates uncontrollably until everything smears into motion blur.
It stops instantly on one centered line of text.

### 2. UI Windows Overload (Pop-Up Storm)
Clean desktop-style UI.
One notification pops up.
Then another.
Then 20.
Then 200.
They fill the screen rapidly until the entire screen becomes a wall of overlapping windows.
Final frame: all collapse into a single centered message.

### 3. Loading Bar Breakdown
A minimal loading bar UI fills from 0% to 100%.
But every time it reaches 100%, it glitches and resets.
After the 4th or 5th loop, the bar breaks apart into pixels, revealing a clean headline underneath.

### 4. Tab Switching Frenzy
A modern browser UI switching tabs at high speed.
Each tab shows a different interface:
news → charts → email → dashboard → music player → AI tool → social feed.
The tab bar stays fixed while the UIs flash beneath.
Ends on a specific tab.

### 5. System Boot-Up Sequence
Black screen → minimalist white system logs appear line by line.
Logs get faster.
Then faster.
Then illegible.
Suddenly: full stop.
Screen clears and shows one centered phrase.

### 6. Split-Screen UI Tesselation
Start with a single clean UI.
The screen splits into two mirrors.
Then four.
Then sixteen tiles, each showing a slightly different UI animation inside.
They zoom back into one final screen.

### 7. File Transfer Warp
A file transfer UI shows a progress bubble moving along a line.
As it moves, the background UI keeps morphing behind it — from drive folders → chat apps → analytics → social media → terminal.
The file bubble arrives at the end and triggers the final reveal.

### 8. Radar Scanner Interface
Circular radar UI sweeps around the screen.
Every time it hits a "blip," the UI flashes to a new screen (website, dashboard, snippet).
Each sweep = new layout.

### 9. Password Attempts UI Lockout
Login UI.
Typing error.
Incorrect password.
Retry.
Incorrect.
The UI shakes, glitches, distorts.
Countdown lockout appears.
Then the whole screen melts into the central message.

### 10. Heatmap Bloom Animation
Start with a bland grey UI.
Suddenly color-coded heatmap blobs begin blooming across the screen — reds, yellows, greens.
They intensify until they take over the whole frame.
Heatmap fades to white → final text appears.

### 11. Interactive Toggle Switch Story
A single toggle switch UI.
You turn it on → screen becomes bright.
You turn it off → UI goes dark.
Toggle becomes hyper sensitive → screen flashes rapidly between states.
Toggle snaps off → message appears.

### 12. AR Overlay Scan
Camera-style UI scanning across a webpage or text.
As the scanline moves, data overlays appear: bounding boxes, labels, stats.
At the end of the scan, all overlays merge to form the headline.

### 13. Typing Interface Takeover
A blank UI with a typing cursor.
Words start appearing.
Then correcting.
Then overwriting themselves.
Then rewriting rapidly.
Text accelerates until the whole screen fills with layered typing.
Everything deletes itself → single line stays.

### 14. Widget Reconfiguration
Dashboard full of draggable widgets.
The widgets automatically move, rotate, resize, reorder — almost like they're alive.
At the end, they snap into one perfect centered shape/message.

### 15. Map Zoom-In/Out Frenzy
A digital map UI rapidly zooms in: continent → country → city → street → building → room.
Each zoom reveals new UI overlays.
Then instantly zooms all the way out to universe scale.
Ends on minimalist final text.

## Section 2: Liquid & Fluid Transitions

### 1. Liquid UI Morph (Soft Shape Transformation)
A clean UI card sits in the center.
Instead of cutting, it melts into a new layout — corners stretch, panels dissolve, buttons melt like soft silicone and reform into new positions.
The whole interface morphs like a liquid blob, staying smooth and organic.
Think: Apple VisionOS meets molten glass.

### 2. Fluid Page Curl (Digital Water Ripple Transition)
A digital webpage ripples like water when touched.
A single tap sends out waves that distort the UI.
Each wave "pulls up" the next interface underneath, like unveiling layers of a liquid screen.
Perfect for a calm but futuristic vibe.

### 3. Ink Flow Reveal
Dark digital ink slowly flows across the screen, covering the UI.
As it passes, the underlying interface changes shape, color, and layout.
Feels like UI elements are being painted into existence.
Very premium, very art-house.

### 4. Liquid Glass Panel Slide
Imagine glass panels sliding across the screen, but instead of rigid movement, they move like viscous gel — bending, stretching, and flowing into place.
Cards "settle" with soft elasticity, like pressing silicone.

### 5. Bubble Interface (Surface Tension Animation)
UI buttons and cards act like bubbles:
– when one expands, nearby elements squash and stretch
– tapping one creates ripples
– transitions mimic surface tension dynamics
Feels playful and modern.

### 6. Melt-to-Text Transition
A UI screen melts downward like hot liquid metal.
As it drips away, the background solidifies into one crisp headline.
Smooth, slow, hypnotic.

### 7. Ocean Wave Data Visual
Charts animate like waves — flowing up and down with smooth sine curves instead of sharp lines.
The camera slowly glides over these "data waves," and each crest reveals a new value or headline.
Elegant, flowing, mesmerizing.

### 8. Liquid Lens Zoom (Refraction UI)
A floating, liquid-like glass lens drifts across the screen.
Wherever it passes, the UI refracts and morphs into a new version — as if the lens is revealing alternate UIs beneath the surface.
Think water droplet meets high-end HUD.

### 9. Soft Gel UI Ripple Cut
Each transition starts with a gel-like ripple in the center.
As the ripple expands, the UI behind it smoothly morphs into the next layout.
No cuts, no jumps.
Just pure fluid transitions.

### 10. Digital Lava Lamp Interface
The UI is structured by fluid blobs that drift around like a lava lamp.
As the blobs merge, split, and float, each creates a mask revealing parts of the next interface.
Abstract, artistic, insanely smooth.

## Section 3: 3D & Dimensional Effects

### 1. Particle Warp Tunnel (Data Particles)
Tiny glowing particles swirl inward, forming a tunnel.
Camera flies through at high speed.
Particles rearrange into shapes → grids → text → dissolve.
Everything feels smooth, flowing, dimensional.

### 2. Morphing 3D Typography
A bold 3D title appears.
Letters stretch, twist, melt, inflate, deflate — like liquid metal morphing.
Final morph locks into a clean, crisp type.
Think Nike / Apple product teaser vibes.

### 3. Ribbon Flow Animation
Thin metallic ribbons (like liquid silk) flow through the screen.
They twist, fold, and weave together to reveal text or a logo.
Elegant, smooth, premium.

### 4. Neon Line Tracing Paths
Electric neon lines trace across a dark background, building shapes and structures.
Paths criss-cross, form circuits, maps, networks.
Everything resolves into a final mark or message.

### 5. Fractal Zoom (Infinite Depth)
Camera zooms infinitely into a fractal-like 3D structure.
Colors and forms shift as you zoom.
At the deepest point, everything collapses into a single phrase.
Cinematic, hypnotic, dynamic.

### 6. Kinetic Typography Burst
Words move aggressively with the beat:
sliding, stretching, exploding, shrinking.
One word builds another → transforms → overlays.
Ends with one impactful line.
Fast, bold, Gen Z energy.

### 7. 3D Liquid Orb Reveal
A floating liquid orb spins in zero gravity.
As it rotates, reflections form images or text inside its surface.
The orb collapses into liquid and splashes into the final reveal.
Perfect blend of fluid + sci-fi.

### 8. Camera Through Shattered Glass
Camera travels smoothly through floating glass shards in 3D space.
Reflections show glitchy hints of text and images.
At the end, the shards snap together to form the final frame.

### 9. Soft Gradient Blobs Dance
Colorful, soft, fluid blobs move like jelly under slow motion.
They merge, separate, swirl around each other.
They mask in/out sections of text and UI.
Very "modern motion design."

### 10. Smoke Simulation Text Reveal
A cloud of digital smoke rolls across screen.
As it moves, it reveals or erases text.
Smooth, atmospheric, premium.

### 11. 3D Wireframe Build-Up
A wireframe grid builds from nothing.
It expands, bends, and constructs a 3D object or scene.
Finally, the wireframe fills itself with solid color.
Technical and crisp.

### 12. Shape Explosion → Reformation
Flat geometric shapes explode outward.
Each accelerates into motion and flies across screen.
Then they snap back together, forming text or a symbol.
Snappy, rhythmic, satisfying.

### 13. Hologram Scan Lines
Horizontal scan lines sweep through the screen like a hologram booting up.
Each pass reveals new details, textures, or text.
Final frame feels like a sci-fi HUD lock-in.

### 14. Metallic Cloth Simulation
A piece of metallic fabric flows in slow motion.
It waves, ripples, bends like silk.
At the end, it drapes itself into the outline of the headline.

### 15. 3D Liquid Droplet Camera Flythrough
Camera dives into a slow-floating droplet.
Inside, visuals warp like entering another dimension.
You emerge out the other side into the next scene.
